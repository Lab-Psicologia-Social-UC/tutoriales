Presentacion

Este documento reune distintos tutoriales y recursos que sirven de guia al realizar diferentes analisis estadisticos recurrentes en el trabajo del Laboratorio. Los objetivos de esta recopilacion son simplificar la produccion de analisis e informes al reunir un cuerpo de codigos reusables, facilitar la transferencia de conocimiento y la actividad colaborativa dentro y fuera del Laboratorio y fomentar la reproducibilidad de la investigacion desarrollada en este.

Para reproducir el codigo presentado en este documento, se necesita una version reciente de R y RStudio, instalar los paquetes especificados en cada guia y tenerlos actualizados.

Para compilar este documento localmente:

  1. Clona o descarga el repositorio asociado y descomprime el archivo .zip.
  2. Abre R Studio.
  3. Instalar el paquete bookdown
  4. Localiza la carpeta del repositorio.
  5. Abre el archivo index.Rmd y haz click en Build Book en la pestaña Build. Alternativamente, ejecuta:
install.packages('bookdown') # instalar bookdown
bookdown::render_book("index.Rmd") # para compilar
browseURL("_book/index.html") # para ver
